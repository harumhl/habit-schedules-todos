# regression-on-stock-market

UI: React.js
Running server: Heroku
Backend: Python
Database: Firebase

React-app
$ npm start
$ npm run build
move the build folder content to public

Heroku (https://devcenter.heroku.com/articles/getting-started-with-python)
$ git add .
$ git commit
$ git push heroku master (in python folder)
$ heroku open

Firebase
$firebase deploy
